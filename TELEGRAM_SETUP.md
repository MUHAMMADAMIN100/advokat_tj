## ⚙️ Настройка Telegram бота

### Шаг 1: Создайте бота
1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Введите имя бота (например: "Advokat TJ Bot")
   - Введите username бота (должен заканчиваться на `bot`, например: `advokat_tj_bot`)
4. **Сохраните токен**, который даст вам BotFather
   - Он выглядит примерно так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

### Шаг 2: Получите Chat ID
1. Откройте бота в Telegram (@TM_Developerr)
2. Отправьте ему любое сообщение (например: `/start`)
3. Откройте в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
   Замените `<ВАШ_ТОКЕН>` на токен из Шага 1
4. Найдите `"chat":{"id":` - это ваш Chat ID
   - Он выглядит примерно так: `123456789`

### Шаг 3: Обновите код
Откройте файл `src/App.jsx` и замените:

```javascript
const TELEGRAM_BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"; // Ваш токен от BotFather
const TELEGRAM_CHAT_ID = "YOUR_CHAT_ID_HERE"; // Ваш Chat ID
```

На:

```javascript
const TELEGRAM_BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"; // Ваш настоящий токен
const TELEGRAM_CHAT_ID = "123456789"; // Ваш настоящий Chat ID
```

### Шаг 4: Проверьте работу
1. Запустите проект: `npm run dev`
2. Откройте сайт
3. Заполните форму и отправьте
4. Проверьте, что сообщение пришло в Telegram
